# Alianci Cleaning

Cleaning services landing page with booking system, auth, and admin dashboard.

Built with [Vite](https://vitejs.dev/), [shadcn/ui](https://ui.shadcn.com/), and [Tailwind CSS](https://tailwindcss.com/).

## Docs

| Reference | Description |
| :-------- | :---------- |
| `CLAUDE.md` / `AGENTS.md` | Project context and rules for AI agents |
| `.claude/docs/PROJECT_STRUCTURE.md` | Folder layout, naming conventions, and where to add new files |
| `.claude/docs/TECH_STACK.md` | Full tech stack |
| `.claude/docs/BUSINESS_LOGIC.md` | Business rules, entities, routes, and flows |
| `.claude/docs/STYLE_GUIDE.md` | Color palette, gradients, component patterns |
| `.claude/docs/SCHEMA_CURRENT.md` | Current Supabase schema |

## Commands

| Command | Action |
| :------ | :----- |
| `pnpm install` | Install dependencies |
| `pnpm dev` | Start local dev server at `localhost:5173` |
| `pnpm build` | Build production site to `./dist/` |
| `pnpm preview` | Preview production build locally |
